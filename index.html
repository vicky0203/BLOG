This is a personal blog which I want to introduce myself and record my life stories. It shows my resume, my universities and my journey from my hometown to HONG KONG, TAIWAN and etc.

All pages' development is based on Bootstrap. I try to use various special effects of CSS3 and Javascript to make my web pages seems more interesting and readable. And I had pay attention to the responsive design for mobile view.

Meanwhile, there are still many areas for improvement. For example, 1. My resume data load by AJAX, but there is no server support for this project, all files are locally run on PC, AJAX are nor support, so I used a trick here to load json data. 2. Some JS function need to be improved to make the webpage more soomthly. 3. the responsive design of the whole website need more test to better support for various of screen size.

I strongly recommend you to browse my blog by downloading the zip. file and run on PC. Also, you can click the link (BLOG PREVIEWS) and check it online, but some CSS or JS effect may be blocked by the online system.

这是一个介绍我个人以及生活的个人博客。 博客中展示了我的个人简历，我的大学以及我从家到香港，台湾，成都等地留学旅行的经历。

博客的所有页面开发都以Bootstrap为基础框架。 同时，我也使用了多样的CSS效果和原生JS控制DOM的动态效果来使得网页更加生动，更吸引读者。 此外，在网页设计开发时也注重移动端布局的设计， 实现了响应式设计。

此项目仍有很多值得改进研究的空间。 比如， 1. 个人简历的数据加载应使用AJAX技术以实现中英文切换，但由于没有服务器支持项目在本地运行，所以使用了一些技巧来加载JSON数据。 2. 一些JS控件还有改进空间以使得整个网页性能更加流畅。 3. 网页响应式布局对多种屏幕尺寸的支持，需要更多的测试及改进。

建议您下载整个项目文件在本地进行浏览以达到最佳效果。 您也可以点击连接 (浏览博客)在线查看， 但一些CSS和JS效果可能由于系统原因加载会出现问题，从而导致某些效果功能缺失。

感谢您的阅读！
